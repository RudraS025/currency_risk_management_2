All currency rates are fetched in real-time from multiple sources including FxRatesAPI and central bank feeds.

The system automatically updates rates and calculations every 30 seconds for live market conditions.

📋 Main System Tabs

📊 Dashboard 📄 Contracts 📈 Analytics 📋 Reporting

1. 📊 Dashboard Tab

Features:
  • Real-time currency rates for all major pairs
  • Live market data updates
  • Currency pair selection and monitoring
  • Market trend indicators

    • 2. 📄 Contracts Tab

    Contract Management:
  • Create new FX forward contracts
  • Edit existing contracts
  • View contract details and performance
  • Track hedge effectiveness
    • Creating a New Contract:
  • Select currency pair (e.g., USD/INR, EUR/USD)
  • Enter contract details (amount, maturity date)
  • System automatically calculates forward rate using IRP
  • Review and confirm contract creation

    • 3. 📈 Analytics Tab

    P&L Analysis:
  • Daily P&L attribution by contract
  • Mark-to-market calculations
  • Historical performance tracking
  • Risk exposure analysis
    • Available Metrics:
  • Unrealized P&L per contract
  • Total portfolio exposure
  • Risk metrics (VaR, Greeks)
  • Performance attribution

    • 4. 📋 Reporting Tab

    Risk Reports:
  • Comprehensive risk dashboards
  • Exposure analysis by currency
  • Compliance reporting
  • Export capabilities

    • 🔧 Key Features

    💱 Interest Rate Parity (IRP)

    Precise profit/loss calculations based on market movements and time decay.
